##Week5 –> 8 review

###Web前端

HTML标签 - 承载内容 - content
- 文本标签：h1~h6 / p / br / hr
- 图像标签：img / figure / figcaption
- 列表标签：ul / ol / dl
- 超链接标签：a（页面链接、锚链接、功能链接）
~ href / target（_self, _blank, _top）
- 表格标签：table / tr / th / td / thead / tbody / tfoot / caption
~ rowspan / colspan
- 表单标签：form / input（text, password, radio, checkbox, file, button, submit, reset, image, hidden, range, date, email,
number, tel）/ textarea / select / option
~ action / method / enctype
- 其他标签：div / span / iframe / audio / video / canvas / header / footer / nav / article / aside

CSS - 渲染页面 - display
- 位置：内嵌样式表（style属性）/ 内部样式表（style标签） / 外部样式表（link标签）
- 选择器：通配符选择器 / 标签选择器 / 属性选择器 / 类选择器 / ID选择器 / 父子选择器 / 后代选择器 / 兄弟选择器 / 相邻兄弟选择器 / 并列选择器 / 伪类 / 伪元素
- 样式属性：文本 / 字体 / 背景 / 边距 / 定位 / 边框
- Box Model（盒模型）
- 如果一个元素被多个选择器渲染那么不冲突的样式会叠加，冲突的样式按照：重要性原则、具体性原则、就近原则进行选择

JavaScript - 交互式行为 - behavoir
- ECMAScript（语法规范）- ES6
- BOM - 浏览器对象模型 - window
- alert / prompt / confirm / close
- setTimeout / setInterval
- location / history / screen / navigator
- DOM - 文档对象模型 - document - 对页面元素进行添加、删除、编辑、查找等一系列操作
- getElementId / getElementsByClassName / getElementsByTagName / querySelector / querySelectorAll
- textContent / innerHTML
- document.createElement / appendChild / insertBefore / removeChild
- children / parentNode / previousElementSibling / nextElementSibling
- addEventListener / removeEventListener

jQuery - Write Less Do More - $
1. $函数的参数是一个函数 - 页面加载完成后要执行的回调函数
2. $函数的参数是一个选择器字符串 - 获取元素对应的jQuery对象
3. $函数的参数是一个标签字符串 - 创建元素并获得对应的jQuery对象
4. $函数的参数是一个原生JS对象 - 将原生JS对象转换成jQuery对象
- jQuery对象常用方法：https://www.runoob.com/manual/jquery/

Vue.js
- 前端渲染（后端服务器通过API接口提供JSON格式的数据，前端页面通过异步请求获得数据，再通过Vue将数据渲染到页面上） - 前后端分离开发 - https://cn.vuejs.org/

###Linux

常用命令
- 文件操作：pwd / cd / ls / mkdir / rm / cp / mv / touch / cat / head / tail / more / less / uniq / sort / wc / cut / paste / file / chmod / chown / chgrp / gzip / gunzip / xz / tar / find / grep
- 网络操作：wget / ssh / scp / sftp / ping / ifconfig / ip / netstat / firewall-cmd / route
- 用户管理：useradd / userdel / passwd / su / sudo / visudo / write / wall / mesg / logout / exit / w / who / whoami / last / lastb
- 进程管理：ps / pstree / top / kill / pkill / killall / jobs / fg / bg / nohup
- 系统管理：reboot / shutdown / init / sar / free / df / iostat
- 计划任务：at / atq / atrm / crontab
vim
- 命令模式：移动光标 / 删除 / 复制 / 粘贴 / 录制宏
- 编辑模式
- 末行模式：配置vim / w / q / sp / vs / 查找 / 替换
服务的安装和配置
- 包管理工具：yum / rpm / apt（Ubuntu）/ deb（Debian）
~ search / install / erase / info / update / list installed
~ -ivh / -e / -qa
- 源代码构建安装：gcc / make
- 下载二进制：配置环境变量即可用

systemctl start / stop / restart / status / enable / disable 服务名
service 服务名 start / stop / restart

运维相关
- 运维相关的命令和工具（Ansible / Negios / Zabbix）
- Shell脚本编程（组合命令+选择结构+循环结构+函数）
- Python程序

数据库

- 关系型 - SQL（DDL / DML / DCL）- MySQL - pymysql
